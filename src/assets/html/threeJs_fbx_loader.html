Three.js使用之fbx模型动画控制

three.js 

因为公司项目需求，需要在web页面中显示模型文件，并能够控制模型文件分段、分次播放。所以入了three.js这个坑。

先上点学习资料：
webGL中文网
很不错的网站，里面涉及了three.js从初级到高级的教学资源。如果你是新手，不要错过，一定好好看。

three.js官网
官网，不多说，都知道是干啥的。英语好的小伙伴好好啃啃官网，能少走不少弯路，我也是在官网中了解到Animation Clips的概念，才另辟蹊径完成模型动画分段调用。英语不好的也可以看three.js中文文档，虽然翻译会存在误差，不过也比没得看好。

three.js GitHub
这个非常给力、非常非常给力、非常非常非常给力……为啥呢？因为它里面存放了大量的demo。不知道该如何入手的小伙伴可以多看看demo，学习一下里面的使用方法。(项目目录下启动本地服务器，在examples目录)

模型文件

既然是模型文件引入，必不可少的当然是模型资源文件。学过3D的小伙伴们知道，3D模型文件的格式那是相当的多，但并不是所有的文件都能够实现web端调用并进行分段播放的。对于这一点，官网是这么说的：

Note that not all model formats include animation (OBJ notably does not), and that only some three.js loaders support AnimationClip sequences. Several that do support this animation type:

大概意思就是，js支持的动画类型有以下这几种：

THREE.JSONLoader
THREE.ObjectLoader
THREE.BVHLoader
THREE.ColladaLoader
THREE.FBXLoader
THREE.GLTFLoader
THREE.MMDLoader
THREE.SEA3DLoader

这里官网还加了一句注意事项，用于提醒动画制作的伙伴们。

Note that 3ds max and Maya currently can't export multiple animations (meaning animations which are not on the same timeline) directly to a single file.

就是说3D MAX和Maya是不支持将多个动画直接导出单个文件。所以，我这里使用的模型包是使用blender导出的。

废话不多说，进入正题。

经过一段时间的摸索，初步掌握了模型文件的引入和控制。现将操作步骤及方法总结如下(以下为关键代码，文章底部付完整代码仓库地址)：

我这里使用的是fbx格式模型

  1. 1 // model 
  2. 2 var loader = new THREE.FBXLoader(); 
  3. 3 loader.load('model/GT_an_8.fbx', function (object) { 
  4. 4 //这个object是个很好玩的东西，控制台打印后你可以在里面的animations里看到模型中的分段动画及持续时间 
  5. 5 object.mixer = new THREE.AnimationMixer(object); 
  6. 6 mixers.push(object.mixer); 
  7. 7 curObject = object; 
  8. 8 //这一步是将模型缩放，默认是(1,1,1)，我的模型太大了，所有都缩放了0.5 
  9. 9 curObject.scale.set(0.5, 0.5, 0.5); 
  10. 10 
  11. 11 //这里是设置动作片段循环次数，根据demo，我了解到loop的值分为三种'2000'(执行一次停止)，'2001'(循环执行)，'2002'(执行一次后回退执行并循环执行) 
  12. 12 curObject.mixer.clipAction(curObject.animations[3]).loop = +'2200'; 
  13. 13 //记住这个方法，它就是播放动画片段的，animations[3]就是播放第四段动画片段 
  14. 14 curObject.mixer.clipAction(curObject.animations[3]).play(); 
  15. 15 
  16. 16 //duration：动画片段持续时间 
  17. 17 //这里实现的是将动画片段4重复播放 
  18. 18 setInterval(function () { 
  19. 19 curObject.mixer.clipAction(curObject.animations[3]).reset(); 
  20. 20 }, curObject.animations[3].duration * 1000); 
  21. 21 scene.add(object); 
  22. 22 
  23. 23 //补充一个方法，stop，字面意思，停止播放动画 
  24. 24 // curObject.mixer.clipAction( curObject.animations[3] ).stop(); 
  25. 25 });
